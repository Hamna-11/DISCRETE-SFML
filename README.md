# 📐 Discrete Structures Visualizer | FAST University

A fullscreen, interactive **C++ + SFML** desktop application that brings Discrete Mathematics concepts to life through a polished GUI. Built as a semester project at **FAST-NUCES**, this tool lets users explore and simulate core Discrete Structures topics using a real university database of students, faculty, courses, and rooms.

---

## ✨ Features

| Module | Description |
|---|---|
| 🔢 **Sets Module** | Union, intersection, difference, and subset operations on course/student sets |
| 🔗 **Relations Module** | Checks reflexivity, symmetry, transitivity, and equivalence relations |
| 🧠 **Logic Module** | Propositional logic with truth table generation and rule evaluation |
| 🔄 **Functions Module** | Injective, surjective, and bijective function verification |
| 📊 **Combinations Module** | Permutations, combinations, and counting principles |
| 🧮 **Induction Module** | Step-by-step mathematical induction proofs |
| ✅ **Proof Module** | Direct, contrapositive, and contradiction-based proofs |
| 📅 **Scheduling Module** | Course prerequisite ordering using Topological Sort |
| ⚡ **Benchmark Module** | Performance testing across algorithmic operations |
| 🧪 **Unit Test Module** | Automated tests for all modules |
| 🔍 **Consistency Checker** | Validates logical consistency across the university database |

---

## 🛠️ Built With

- **C++14** : core logic and data structures
- **SFML 2** : fullscreen GUI, rendering, and event handling
- **Visual Studio 2022** : project setup (`.sln` / `.vcxproj`)

---

## 🚀 Getting Started

### Prerequisites
- Windows 10/11
- Visual Studio 2022 (with C++ Desktop Development workload)
- SFML 2 (already included in `/lib` and `/include`)

### Running the Project

1. Clone or download this repository
2. Open `SFML1.sln` in Visual Studio
3. Set build to **x64 | Release** (or Debug)
4. Press **F5** to build and run

> All required SFML `.dll` files are already included in `x64/Release/` — no extra setup needed.
